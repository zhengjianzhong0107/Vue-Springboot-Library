# 图书馆管理系统_毕设_1009

#### Description
图书馆管理系统_毕设, 主要技术为SpringBoot、Mybatis-Plus、MySQL、Vue3、ElementPlus,管理员模块：注册、登录、书籍管理、读者管理、借阅管理、借阅状态、修改个人信息、修改密码

读者模块：注册、登录、查询图书信息、借阅和归还图书、查看个人借阅记录、修改个人信息、修改密码
